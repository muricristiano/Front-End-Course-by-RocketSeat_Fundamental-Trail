Podemos propor alterações para projetos de outros usuários

Podemos a todo momento editar arquivos de texto de outras pessoas e em vez de termos a opção de salvar, temos "Propor alterações"

            É como criar "working directorys" nos arquivos de outras pessoas no github.


Essa alteração vai para o "PULL REQUEST" do usuário

para ele verificar e aceitar/recusar a alteração!


# Forks

   É proveniente do fluxo de trabalho do GitHub.

   Toda vez que editamos um arquivo de outra usuário, ele cria um fork, que significa que ele traz um "clone" do repositório daquele usuário, um repositório de fora (um fork). Para ser trabalhado ao longo do tempo, etc. E proposto quando quisermos. Ou excluído quando quisermos.
