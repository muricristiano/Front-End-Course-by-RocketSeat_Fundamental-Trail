Display Inline vs Display Block

Inline

Elementos que permanecem na mesma linha como: <a>, <strong>, <span>, <em>, etc.

Block

Elementos que quebram, iniciam uma nova linha, e ocupam a linha inteira, display blocks: <p>, <div>, <section>, e todos os headings como: <h1> <h2>, etc...
